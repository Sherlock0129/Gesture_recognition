# 手势识别项目计划 (基于 CV & MediaPipe)

考虑到您希望使用**计算机视觉**且**数据集尽可能小**，最佳方案是：**MediaPipe + 自定义轻量级分类器**。
MediaPipe 可以直接从图像中提取出 21 个手部关键点（Landmarks），我们将这些关键点坐标作为特征进行分类，而不是直接对图像像素进行深度学习训练。这样不仅计算量极小，而且几十或几百条数据就能训练出一个精度很高的模型。

## 阶段一：环境搭建 (Environment Setup)
- [ ] 创建 Python 虚拟环境（可选但推荐）
- [ ] 安装依赖库：`opencv-python`, `mediapipe`, `scikit-learn`, `numpy`, `pandas`

## 阶段二：数据采集与预处理 (Data Collection)
- [ ] 编写数据采集脚本 `collect_data.py`：
  - 调用电脑摄像头。
  - 使用 MediaPipe 检测手部并提取 21 个关键点坐标。
  - 提示用户做出不同手势（如：石头、剪刀、布、OK 等），将提取的坐标数据（转换为相对距离以消除位置影响）保存到 CSV 文件中。
  - *说明：这是一个非常小的数据集，每张图片只提取 63 个浮点数 (21个点 x 3维)*。

## 阶段三：模型训练 (Model Training)
- [ ] 编写训练脚本 `train_model.py`：
  - 读取 CSV 中的坐标数据集。
  - 使用轻量级机器学习算法（如 SVM、随机森林 或 KNN）训练分类器。
  - 评估模型准确率，并保存训练好的模型文件（如 `.pkl` 文件）。

## 阶段四：实时推理与展示 (Real-time Inference)
- [ ] 编写实时识别脚本 `recognize_gesture.py`：
  - 开启摄像头实时捕获画面。
  - 使用 MediaPipe 提取当前帧的手部关键点。
  - 将关键点输入到训练好的模型中进行预测。
  - 在视频画面上绘制关键点骨架，并显示识别出的手势名称。
